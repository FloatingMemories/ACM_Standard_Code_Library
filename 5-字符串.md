# 字符串

## 最小表示法

- 寻找一个字符串的循环同构串中最小的那一个，输出偏移量

```cpp
int min_string(string s){
	int k = 0, i = 0, j = 1, n = s.length();
	while (k < n && i < n && j < n) {
		if (s[(i + k) % n] == s[(j + k) % n]) {
			k++;
		} else {
			s[(i + k) % n] > s[(j + k) % n] ? i = i + k + 1 : j = j + k + 1;
			if (i == j) i++;
			k = 0;
		}
	}
	return min(i, j);
}
```
